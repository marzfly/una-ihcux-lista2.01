## Como eu fiz o CANTINOTRON 3000
1. Conversão de variáveis string em inteiro para possibilitar a leitura do input do usuário
2. ArrayList para armazenar os pedidos
3. Métodos de leitura, visualização e saída
4. - Leitura: mantinha o usuário num loop while enquanto ele não digitasse o código correto. Quando o código correto fosse digitado, ele saía do loop e adicionava o código ao array anteriormente mencionado
   - Visualização: mostrava quais produtos o usuário tinha no seu carrinho. Caso ele quisesse remover algum produto, digitava o código de acordo com o que aparecia na tela (i + 1) e o sistema retirava 1 para ficar de acordo com a posição.
   - Saída: exibia uma mensagem de agradecimento ao cliente e fechava o programa
5. Parte principal:
   - Menu: informando as ações disponíveis para o usuário
   - Loop enquanto a condição de saída não fosse obtida, possibilitando o usuário repetir a ação quantas vezes quisesse
